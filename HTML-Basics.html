Basic HTML Elements

Working with Text

For paragraphs, the p element is used. To make a character or text bold or italicized, the strong or em elements are used, respectively.

Links and Images

Links use the anchor element a and attribute href; can either use absolute or relative links. Aboslute links are links to pages on other websites, while Relative links are links to other pages within our own website. The latter does not include a domain name, as it assumes the domain name to be the same as the page the link was created on. For example, href="pages/about.html" is a relative link and assumes that the domain is index.html where the link is created

Images use element img and attributes src to provide image source, either locally or through a url, and alt which provides a short description of the image in case it doesn't load in properly

Lists

Lists use either ul or ol for either unordered or ordered lists. List elements are then indented under ul or ol with the li element.
